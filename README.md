# icecreamplcsimulator
PLC Technical Assessment Simulator is ready — a complete, single-file, production-grade HTML application. Here's what's inside:
PLC Simulator (4 Modules)
Live Digital I/O Panel with 6 toggle inputs and 4 animated LED outputs responding to real ladder logic (seal-in, interlocks, E-Stop master cut)
Live Ladder Diagram that updates in real-time showing energized contacts and coils
Animated Scan Cycle Visualizer cycling through all 5 phases with live ms timing
Fault Injection Mode for 4 fault types (sensor, CPU, PROFINET, temperature) with visual alarms and fault log
Assessment Engine
20 technical multiple-choice questions across PLC, timers, PID, PROFINET, food safety, OEE, and electrical troubleshooting
5 practical Serbia ice cream plant troubleshooting scenarios
45-minute countdown timer with progress bar
Auto-grading with correct answer reveal and explanations
Results Dashboard
Score cards for Total / Technical / Scenario scores
Pure JavaScript performance bar chart with pass-line overlay
Grade system: Deployment Ready / Technically Competent / Needs Improvement / Not Ready
Printable deployment certificate
Workbook Features Added:
Candidate Info Form — Full Name, Date, Position, Deployment Location, Assessor, auto-filled Score (from session storage if exam was taken), unique Assessment ID generator
Cover Page — Formal industrial document layout with double-border, candidate metadata grid, facility branding, and confidentiality stamp
Section A — All 20 MC questions formatted as bubble-sheet exam questions in a clean two-column grid with numbered question blocks
Section B — All 5 troubleshooting scenarios with highlighted situation boxes, options, and justification lines
Section C — 3 ladder logic drawing exercises with dot-grid drawing areas, example rung templates, and per-question scoring
Section D — 4 short-answer questions (scan cycle, interlocking, PID, food safety/HACCP) with answer lines
Answer Key Toggle — Checkbox reveals/hides correct answers and model answers throughout all sections
Scoring Summary Page — Full mark table, 4-tier deployment rating grid (auto-highlighted if exam taken), assessor comments area, and three signature lines
Print CSS — Hides all simulator UI, shows only workbook, A4 formatting, section page breaks, print-color-adjust for backgrounds, B&W compatible
